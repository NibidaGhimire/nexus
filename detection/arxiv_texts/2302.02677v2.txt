M.F. Newman, E.A. O’Brien and M.R. Vaughan-Lee
Abstract. We list those presentations for the groups of order p6for prime p≥7 which are distributed
as part of the SmallGroups library.
In [7] we proved that there are
3p2+39p+344+24gcd( p−1,3)+11gcd( p−1,4)+2gcd( p−1,5)
isomorphism types of groups of order p6for every prime p≥7.
Our determination of these groups diﬀers signiﬁcantly from that cla imed by James [ 6]; some of the
errors in his work are recorded in [ 9]. See [7] for details of checks performed on our determination. We
refer the reader interested in the chequered history of classiﬁca tion of these groups to [ 7].
Our presentations for these groups are available publicly as part of theSmallGroups library [1].
There continues to be demand for a human-readable description of the groups. One possible approach is
to apply the Baker-Campbell-Hausdorﬀ formula to the correspond ing Lie ring presentations listed in [ 10].
Here we provide group presentations explicitly: the parametrised p resentations listed in Table 1mirror
those stored in the realisations of SmallGroups distributed in GAP[4] andMagma [2]. An electronic
version of the latter code is available at [ 8].
The groups are classiﬁed into 43 isoclinism families labelled Φ ifor 1≤i≤43. Invariants for each
family are tabulated in [ 6]; these include central quotient, derived group, lower central se ries structure,
conjugacy class structure, and number of irreducible character s.
The ordering of the rows in Table 1is based on the ordering of the isoclinism families to which the
groups belong. We follow the ordering of these families given by Easte rﬁeld [3]. This is ﬁrst by rank. The
ordering within families follows that of [ 3] – except for Φ 19which follows [ 6].
The families of rank 6 are Φ 11to Φ43. These are ordered by decreasing size of the centres. With the
exception of Φ 19, each listed presentation has generating set {α1,...,α 6}taken from a composition series
of the group which reﬁnes its upper central series, so α1is always central. The relation set consists of power
relations and ﬁfteen commutator relations [ αi,αj] =...– we omit those commutator relations which are
trivial. Some of the relations involve ν, the smallest positive integer which is a quadratic non-residue mod
p, and some ω, the smallest positive integer which is a primitive root mod p. For some rows there are one
or two additional parameters. Examples are:
•G(11,14r)whererruns over 1 ,ν;
•G(16,12r)wherer= 1,ω,ω2whenp≡1 mod 3, and r= 1 when p≡2 mod 3;
•G(18,9r)wherer= 1,ω,ω2,ω3whenp≡1 mod 4, and r= 1,νwhenp≡3 mod 4;
•G(21,7rs)forr= 0,1,...,p−1 ands= 1,2,...,(p−1)/2.
The lists of presentations are generated by taking these paramet ers in a ﬁxed order.
